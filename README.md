# OpenWrt Firmware Selector

Another firmware selector for OpenWrt, built with Vue 3 + PrimeVue.

## Live Demo

- [OpenWrt Firmware Selector](http://czbix.github.io/openwrt-firmware-selector/)
- [ImmortalWrt Firmware Selector](http://czbix.github.io/openwrt-firmware-selector/ImmortalWrt/)

## Configuration

See `.env.example` for all available environment variables.
You can copy it to `.env.local` and adjust values for your environment.

For the other brands, you may want to replace files in `/public` folder.

## Build

```sh
pnpm install
pnpm build
```
